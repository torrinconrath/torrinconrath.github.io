Game / 2023

Agent vs Ghosts

A top-down shooter where endless waves of ghosts come for you. Dodge, shoot, collect powerups, and see how long you last before the spawns become impossible.

Python Pygame Canvas Port Ver 1.2

Play

Agent vs Ghosts

ver. 1.2 — survive as long as you can

WASD — move    Arrow Keys — shoot
3 lives    ghosts escalate over time

Controls

  • W A S D Move agent
  • ↑ ↓ ← → Shoot
  • P Pause

Powerups

Lightning

Double movement speed

Cannon

Shoot all 4 directions at once

Bomb

Wipes all ghosts after 5s

Infection

Killed ghosts spawn a zombie decoy

Shield

Absorbs one hit

Star

Invincible, 2x speed, gold omnidirectional bullets

Honey

Cuts all enemy speeds in half

Firecracker

Shoot proximity-triggered rockets
